# UX Patterns Rules

## Trigger
- `**/app/(tenant)/**` directory

## Menu Navigation Patterns

### Category Navigation
- Horizontal scrollable categories on mobile
- Sticky header with search
- Quick filters (vegetarian, spicy, etc.)
- Search with instant results

### Dish Display
- Card grid on desktop (3 columns)
- Single column on mobile
- High-quality image as hero
- Clear price and description
- Quick add button visible

### Order Flow
1. Browse menu → Add to cart
2. Review cart → Modify quantities
3. Add notes/modifiers
4. Confirm order → WhatsApp/checkout

## Interaction Patterns

### Add to Cart
- Tap dish card → Modal with details
- Select modifiers → Add button
- Toast confirmation → Mini cart updates

### Stories Pattern
- Horizontal scrollable stories
- Tap to view story
- Auto-advance after 5s
- Swipe to next/previous

### Search
- Instant search as you type
- Highlight matching text
- Show recent searches
- Filter by category

## Accessibility
- Keyboard navigation for all actions
- Screen reader announcements
- Focus management in modals
- Color contrast WCAG AA
- Touch targets min 44x44px

## Mobile-First
- Bottom navigation for main actions
- Swipe gestures for common actions
- Pull to refresh
- Infinite scroll for long lists
- Sticky CTA buttons

## Loading States
- Skeleton screens for content
- Optimistic UI updates
- Progressive image loading
- Offline indicator
